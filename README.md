# SimFaaS-NSGD: A Serverless Simulator

## Requirements

- Create a python environment
```sh
python -m venv env_name
```
- Activate the environment
```sh
source env_name/bin/activate
```
- Install the requirements
```sh
pip install -r requirements.txt
```
